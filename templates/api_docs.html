API Yetkilendirmesi (Authentication)

Tüm API isteklerinizde kimlik doğrulaması için **X-API-Key** başlığını (Header) veya istek sonuna **api_key** parametresini eklemelisiniz.

Kişisel API Anahtarınız (API Key)

API Uç Noktaları (Endpoints)

GET /api/v1/page

Mevcut kullanıcının sayfa ve blok bilgilerini JSON olarak getirir.

curl -X GET "http://{{ .Host }}/api/v1/page" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}"
POST / PUT /api/v1/page

Sayfa başlığını, biyografisini veya temasını günceller.

curl -X POST "http://{{ .Host }}/api/v1/page" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}" \ -H "Content-Type: application/json" \ -d '{ "title": "Yeni Başlığım", "bio": "Yeni biyografi yazım...", "theme": "emerald" }'
GET /api/v1/blocks

Tüm içerik bloklarını listeler.

curl -X GET "http://{{ .Host }}/api/v1/blocks" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}"
POST /api/v1/blocks

Sayfaya yeni bir blok ekler. (Tip: 'link', 'text', 'social_links')

curl -X POST "http://{{ .Host }}/api/v1/blocks" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}" \ -H "Content-Type: application/json" \ -d '{ "type": "link", "title": "Kişisel Websitem", "content": "https://ornek.com" }'
POST / PUT /api/v1/blocks/update

Belirli bir bloğun içeriğini, başlığını, stilini (style) veya ikonunu (icon) günceller.

curl -X POST "http://{{ .Host }}/api/v1/blocks/update" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}" \ -H "Content-Type: application/json" \ -d '{ "id": 42, "title": "Sosyal Medya", "content": "{\"instagram\":\"kullaniciadi\",\"twitter\":\"kullaniciadi\"}", "style": "brutalist", "icon": "instagram" }'
DELETE /api/v1/blocks?id={id}

Belirtilen ID'deki bloğu siler.

curl -X DELETE "http://{{ .Host }}/api/v1/blocks?id=42" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}"
GET /api/v1/stats

Linklerinizin tıklanma istatistiklerini (clicks) azalan sırada getirir.

curl -X GET "http://{{ .Host }}/api/v1/stats" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}API_ANAHTARI{{ end }}"
