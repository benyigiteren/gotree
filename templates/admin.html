API & Entegrasyon

Yapay Zeka Ajanları, otomasyonlar ve harici uygulamalar için REST API

v1 · REST · JSON
Kişisel API Anahtarınız (X-API-Key)
{{ if .APIKey }}{{ .APIKey }}{{ else }}API Anahtarı bulunamadı.{{ end }}
Tüm isteklerinizde X-API-Key: <anahtarınız> başlığı ekleyin. Base URL:
GET /api/v1/page Kimlik doğrulama gerekli

Hesabınıza bağlı sayfanın tüm bilgilerini ve bloklarını JSON olarak döner.

# Curl örneği
curl -X GET "/api/v1/page" \
  -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<API_ANAHTARI>{{ end }}"
Örnek Yanıt: 
{"page":{"id":1,"slug":"kullanici","title":"Ad Soyad","bio":"Açıklama"},"blocks":[...]}
POST/PUT /api/v1/page Kimlik doğrulama gerekli

Sayfa başlığını, biyografisini, slug'ını veya temasını günceller. Yalnızca gönderilen alanlar güncellenir.

İstek Gövdesi (JSON)
AlanTipAçıklama
titlestringSayfa başlığı
biostringBiyografi açıklaması
slugstringURL kısaltma (benzersiz olmalı)
themestringTema adı (minimal_cream, galaxy_night...)
curl -X POST "/api/v1/page" \
  -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<API_ANAHTARI>{{ end }}" \
  -H "Content-Type: application/json" \
  -d '{"title":"Yeni Başlık","bio":"Yeni biyografi","theme":"galaxy_night"}'
GET /api/v1/blocks

Sayfanızdaki tüm blokları listeler (link, text, social_links, form).

curl -X GET "/api/v1/blocks" -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}"
POST /api/v1/blocks 201 Created

Sayfaya yeni blok ekler. Tipler: link text social_links form

curl -X POST "/api/v1/blocks" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}" \ -H "Content-Type: application/json" \ -d '{"type":"link","title":"Websitem","content":"https://ornek.com"}'
POST/PUT /api/v1/blocks/update

Belirli bir bloğu günceller. Bloğun id'si zorunludur.

AlanAçıklama
id *Bloğun ID'si (zorunlu)
titleYeni başlık
contentYeni içerik / URL
styleStil (standard, brutalist, neon, glass...)
iconİkon adı veya upload URL
curl -X POST "/api/v1/blocks/update" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}" \ -d '{"id":42,"title":"Yeni Başlık","style":"brutalist","icon":"globe"}'
POST /api/v1/blocks/reorder YENİ

Blok sırasını değiştirir. order alanına blok ID'lerini istediğiniz sırayla verin.

curl -X POST "/api/v1/blocks/reorder" \ -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}" \ -d '{"order":[3,1,2,5,4]}'
DELETE /api/v1/blocks?id={id}

Belirli bir bloğu kalıcı olarak siler. Geri alınamaz.

curl -X DELETE "/api/v1/blocks?id=42" -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}"
GET /api/v1/stats GELİŞTİRİLDİ

Sayfa görüntüleme, link tıklama, cihaz kırılımı, ülke ve sosyal medya istatistiklerini döner.

Query Parametreleri
ParametreDeğerlerAçıklama

}
GET /api/v1/form-submissions YENİ

Form bloğunuza gelen tüm iletişim mesajlarını listeler.

curl -X GET "/api/v1/form-submissions" -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}"
Örnek Yanıt: 
{"total":3,"submissions":[{"id":1,"name":"Ali","email":"ali@mail.com","message":"Merhaba...","created_at":"..."}]}
DELETE /api/v1/form-submissions Kimlik doğrulama gerekli

Belirtilen ID'ye sahip form mesajını sistemden siler. Query parametresi olarak `id` gönderilmelidir.

curl -X DELETE "/api/v1/form-submissions?id=1" \
  -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}"
Örnek Yanıt: 
{"success":true,"message":"Mesaj başarıyla silindi"}
GET /api/v1/system/stats Kimlik doğrulama gerekli

Sistem çalışma süresi (uptime), sunucu bellek kullanımı, DB dosya boyutu ve toplam veritabanı kayıt istatistiklerini getirir.

curl -X GET "/api/v1/system/stats" \
  -H "X-API-Key: {{ if .APIKey }}{{ .APIKey }}{{ else }}<KEY>{{ end }}"
Örnek Yanıt: 
{"system":{"uptime":"3h5m12s","go_version":"go1.22","ram_used_mb":"12.45 MB","db_size_kb":56},"database":{"total_pages":2,"total_blocks":12,"total_clicks":150},"status":"healthy"}
Bu endpoint'ler kimlik doğrulama gerektirmez. Dış uygulamalar, widget'lar veya herhangi bir üçüncü taraf entegrasyon için kullanabilirsiniz.
GET /api/v1/profile/{slug} AÇIK · AUTH YOK

Herhangi bir Gotree profilini slug ile sorgular. Hassas bilgiler (user_id vb.) gizlenir. CORS başlıkları açıktır.

curl -X GET "/api/v1/profile/{{ if .Page }}{{ .Page.Slug }}{{ else }}kullaniciadi{{ end }}" # API anahtarı gerekmez!
Yanıt: 
{"profile":{"slug":"...","title":"...","bio":"...","theme":"...","avatar_url":"..."},"blocks":[...]}
